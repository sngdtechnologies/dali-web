# Dali Web — Images à fournir

Toutes les images ci-dessous ont aujourd'hui un **placeholder** (illustration SVG maison ou pastille).
Remplace-les au fur et à mesure : dépose le fichier dans `public/` au **chemin indiqué**, coche la case,
puis préviens-moi (ou remplace directement — le code pointe déjà vers ces chemins une fois câblés).

Sources gratuites (usage commercial OK) : **Unsplash**, **Pexels**, **Pixabay** (photos) ·
**unDraw**, **Storyset** (illustrations SVG) · **Lucide**, **Heroicons** (icônes).

## Récap

- **Minimum crédible : 4 fichiers** — logo, 1 photo hero, 1 capture d'app, 1 image OG.
- **Habillage complet : ~10–14 fichiers** — logo (×2), photo hero, 6 captures d'app, image OG, 2 badges stores, quelques logos partenaires, favicon.

## Notes

- Les **captures d'app** = tes propres écrans Dali (export depuis l'app).
- La **photo hero** : Unsplash/Pexels (ex. « person using phone finance / mobile banking »).
- L'**image OG** et un **logo provisoire** : je peux te les générer en SVG aux couleurs Dali (ivoire / forêt / or) en attendant les définitifs.
- Formats web : préfère **WebP** ou **JPG** pour les photos (léger), **SVG** pour logos/icônes, **PNG** pour les captures avec transparence.
